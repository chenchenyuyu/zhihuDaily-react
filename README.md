# zhihuDaily-react
知乎日报react版
技术栈：
* react
* react-redux
* react-router-dom
* axios
* less
* icon : 使用阿里图标库
# Usage

* 项目根目录安装依赖

  npm install

* 浏览器访问localhost:8000

  npm run dev

* 打包编译生成静态资源

  npm run build
